## Model Details

This model is an int4 model with group_size 128 and symmetric quantization of [Qwen/Qwen3.5-2B](https://huggingface.co/Qwen/Qwen3.5-2B) generated by [intel/auto-round](https://github.com/intel/auto-round). Please follow the license of the original model.

## How to Use

### Environment

```bash
uv pip install vllm --torch-backend=auto --extra-index-url https://wheels.vllm.ai/nightly
uv pip install -U transformers
```

### VLLM Usage

```bash
vllm serve Intel/Qwen3.5-2B-int4-AutoRound \
    --host localhost \
    --trust-remote-code \
    --dtype bfloat16
```

## Generate the Model

```bash
auto-round --model_name Qwen/Qwen3.5-2B --bits 4 --iters 200 --output_dir Qwen3.5-2B-int4-AutoRound
```
